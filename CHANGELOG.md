# 共情剧场功能增强 PR

## 概述
此 PR 实现了多项功能增强和 UI 优化，主要包括 AI 发言决策机制、对话分析报告增强、UI 布局优化等。

## 功能变更

### 1. AI 发言决策机制
- **NPC 沉默机制**：AI 现在可以判断何时不适合发言，显示 `......（沉默）`
- **智能发言判断**：用户话语未完时、对话需要停顿时，AI 会选择沉默
- **修改 `src/lib/prompts.ts`**：添加智能发言判断规则
- **修改 `src/app/chat/page.tsx`**：实现消息解析和显示逻辑

### 2. 对话分析报告增强
- **Markdown 渲染**：支持 `**加粗**`、标题、列表等格式
- **提升可读性**：改善报告视觉效果和结构层次
- **修改 `src/components/FeedbackPanel.tsx`**：实现 Markdown 渲染功能

### 3. UI 布局优化
- **首页布局调整**：名字框占 30%，身份框占剩余空间
- **间距优化**：增加输入框间距，改善视觉平衡
- **多行文本**：角色细节框改为自适应高度的 textarea
- **修改 `src/app/page.tsx`**：调整 NPC 输入区域样式

### 4. 交互流程改进
- **手动触发**：AI 发言后停留，需点击切换到下一条
- **修改 `src/app/chat/page.tsx`**：调整消息队列播放逻辑

### 5. 人像显示优化
- **位置调整**：AI 人像显示在左下角
- **生成优化**：避免并发请求导致的频率限制
- **修改 `src/app/chat/page.tsx`**：调整人像容器位置
- **修改 `src/app/page.tsx`**：优化人像生成逻辑

### 6. API 优化
- **重试机制**：增加 API 请求重试间隔，减少频率限制
- **修改 `src/app/api/chat/route.ts`**：增强错误处理

## 技术细节
- 保持向后兼容性
- 无需数据库变更
- 遵循项目现有代码风格
- 优化用户体验
